What each panel means

Ledger
Proof-carrying claims the substrate has learned, each scored by how often evidence has confirmed it (val).
Cost / Cache
Real spend and tokens saved, measured from your session metrics — asserted by nothing, counted from what happened.
Impact
The blast radius of a change: which files a given symbol or file would ripple out to.
Radar
Whether each dependency is current, deprecated, or worth adopting — grouped into adopt / trial / assess / hold rings.
Trends
How each pipeline stage has behaved over the recent window, as a per-stage sparkline of activity.
Memory browser
Search past claims, ranked by relevance × freshness × confidence — the same recall that powers forge deja.
Session timeline
The durable history of claims minted and retracted, carried across sessions.
